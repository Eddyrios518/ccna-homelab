# Lab 01 – Basic LAN Connectivity

## Topology
PC1 → Switch → PC2

## Goal
Configure two PCs on the same network and verify connectivity using ping.

## IP Addressing

PC1
192.168.1.10
255.255.255.0

PC2
192.168.1.11
255.255.255.0

## Test

Ping from PC1 to PC2.

Result:
Successful communication.

## Concepts Learned

- Basic LAN topology
- IP addressing
- Layer 2 switching
- Connectivity testing with ping
